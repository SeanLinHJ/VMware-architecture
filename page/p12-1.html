12-1 設定 isolation address (隔離區位置)


step 1. 選擇 esxi 01 主機


step 2. 設定 DNS 與 gateway


step 3. 將 gateway 設為 192.168.2XX.1

這個 IP 位置即為 isolation address

isolation address 為 LAN 中的一個特定主機，所有的機器都有連到它，通常就是 gateway。但我們是使用 host-only 連線，沒有 gateway，所以 isolation address 改為本機的 IP，esxi 主機皆可以連接到本機，因此才使用本機的 IP 填入 gateway 的位置


step 4. 確認 gateway 是否正確
